# 明日方舟生息演算地图查询

依托于寒冰制作的地图详解，我制作了一个网页进行关键字查询，方便大家使用（纯纯的用爱发电捏）

由于服务器是学生版的所以流量承载力不够大，欢迎大佬 fork 该项目进行再部署

注意！本项目尚且仅对移动端进行了适配哦，pc端想使用的话请用F12开启手机模拟模式
